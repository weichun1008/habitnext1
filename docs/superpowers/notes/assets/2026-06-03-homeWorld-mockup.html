② 越住越溫馨：同一個家的成長

完成習慣 → 家長出胖胖家具。漏天不會掉家具（零懲罰），只是今天沒解鎖新的。

剛搬進來 空屋
還沒完成任何習慣。空地磚 + 一個搬家紙箱。「完成第一個習慣，搬進你的家。」

幾個面向有進度：客廳有沙發、廚房有餐桌、健身角有啞鈴。其他區還是空地磚，等你補完成。

④ 世界選擇器（首次迎賓 + 之後切換，跨 session 共用）

同時只玩一種世界。首次點「世界」入口迎賓選一個（不在註冊強迫）；之後隨時可切、不限頻、零確認。切換後新世界從「共同序章」起步，舊世界不歸零。

⑤ 互動：解鎖 → 待擺抽屜 → 格位佈置

佈置模式（slot-based）

點某區放大 → 進佈置模式。左側「待擺抽屜」列出已解鎖未擺的家具；點家具 → 點房間空格位放入；點已擺的 → 移回抽屜。不做像素級自由拖拉（手機難用），用固定格位：乾淨、確定、好測。

解鎖回饋

完成習慣使某面向跨過門檻 → 解鎖新家具。解鎖即恭喜、不強制擺：自動進「待擺抽屜」第一個空格先放著，之後可自由調整。

空屋起步溫馨小窩理想之家
實作備註：等距房間＝純 SVG（暖棕描邊 + 圓角，零外部資產），確定性佈局（seed = hash(zone+slot)，禁 render 期 Math.random）。家具解鎖純從完成數衍生（零 schema，學旅程世界）；只有「擺位」存 DB（HomeFurniturePlacement，slot-based）。色票重用 CATEGORY_CONFIG（lib 已納入 tailwind content 掃描）。分階段 H1（世界切換基礎＋選擇器，跨 session 共用）→ H2（唯讀房間）→ H3（手動佈置）→ H4（打磨）。
