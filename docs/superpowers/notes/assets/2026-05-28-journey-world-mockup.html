這版怎麼回應你所有的需求

雙引擎

走＝探索（解鎖霧區）、吃＝收集（拍立得插旗）。兩個普世、終身、產生真實資料的行為當底層；其他習慣是「你正在練的功」+ 達人計劃。

不接地圖 API · 不靠步數

世界是純 SVG 風格化插畫，城市級插旗。零地圖成本，web 上也跑得動。步數那種要原生整合的留到 cofit app。

工具乾淨 / 收集分開

「今天」分頁是純清單（產品目的）；「旅程」分頁才是收集世界（留存引擎）。不潑灑、不麻痺、opt-in。

零懲罰

漏一天不會失去任何收集。沒做只是沒長新東西。用「珍惜」拉回來，不是「恐懼」（比 Forest 進步的地方）。

結合現實世界

飲食照片＝你真實生活的回憶（旅行、美食），這是 Pikmin 明信片機制的對應，而且是你們本來就在拍的東西。

達人飛輪

專家簽名徽章＝社交貨幣 + 行銷 + 變現分層。蔡英傑「腸道修復者」、文長安「食安守護者」各有專屬章。

仍待你定的：① 視覺調性（這版是「明亮現代插畫」，也可更暖手作或更冷科技）② 走路要不要真的影響解鎖、還是純象徵 ③ 個人習慣要 (a)純底層 還是 (b)有加成回饋到世界。
