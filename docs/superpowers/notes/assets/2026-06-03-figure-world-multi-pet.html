不一樣的萌寵

小芽(嫩芽) / 奶茶(貓) / 啵啵(史萊姆)，物種不同、個性不同、成長外型不同，但跑同一引擎。

平行存檔

一次專心養一隻（情感集中）；換萌寵＝開新存檔，舊的不刪可切回。新隻可用「連續天數/里程碑」解鎖，給長期動機。

v1 建議

先把小芽一隻 + 三層系統做好，奶茶/啵啵當「之後解鎖」。系統一旦成立，加新萌寵成本很低。

待定：① 新萌寵的解鎖條件（連續天數 / 里程碑 / 季節活動）② 一次只能養一隻、還是可同時養多隻 ③ 是否要讓萌寵也能像之前說的「依習慣組成進化不同形態」（可疊加在物種之上）。
