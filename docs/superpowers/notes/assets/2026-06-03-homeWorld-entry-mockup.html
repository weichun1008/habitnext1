進入居家世界 🚪🏡

從「選世界」到第一次踏進家的進場儀式。四個關鍵畫面：迎賓選擇 → 開門過場 → 首次進入（空屋引導）→ 回訪進入（溫馨的家）。沿用居家世界調性。

這是 進場視覺提案。對應 spec §1.3 世界選擇器。手機框示意，動畫（開門、家具浮現）以關鍵幀呈現。
流程：迎賓 activeWorld=null → 選居家 → 開門過場 ~600ms → 首次：空屋 + CTA ‧‧‧（日後回訪）‧‧‧ 溫馨的家 + 今日提示
STEP 1 · 首次
迎賓選世界
點「世界」入口、activeWorld=null 時出現。居家卡 highlight。底部明說「選前的完成＝共同序章，選哪個都帶著走」——消除「太晚選會虧」的焦慮。
STEP 2 · 過場 ~600ms
開門過場
選定後一段短儀式：木門半開、門縫透暖光 + 家具剪影、鑰匙 🔑 收進口袋。輕、不打斷，給「踏進一個空間」的儀式感（≈ Slice P 沒有的、居家獨有的進場感）。
STEP 3 · 首次進入
空屋 + 搬家引導
空地磚 + 搬家紙箱（即使有序章完成數，第一次也用「歡迎入住」框住）。CTA 把人帶回「今日」完成習慣——遊戲化不取代工具，而是回流動機。
STEP 4 · 之後回訪
溫馨的家 + 今日提示
頂部溫馨度條（取代旅程世界的城市階級）。「今天完成 N 個 → 解鎖 XX」把遊戲化跟今日任務勾起來。主 CTA「進入佈置」進房間擺家具。回訪沒有開門過場（只首次有）。

進場節奏（什麼時候看到什麼）

三種進入情境共用同一個居家畫面，差別在「前面那一下」：

A. 首次選定居家
迎賓選擇 → 開門過場 → 空屋 + 搬家引導。唯一有完整儀式的一次。
B. 平常點「居家」入口回訪
直接進溫馨的家（不開門過場，避免每次都儀式變囉嗦）。頂部溫馨度條 + 今日提示。
C. 從別的世界切回居家
短暫「歡迎回家」淡入（比首次輕），強調「你的家還在、進度沒掉」。呼應共同序章哲學。
原則：儀式感只給首次（開門）。回訪 / 切換要快、不打斷——對齊「工具乾淨、收集分開、不潑灑不麻痺」。零懲罰：回訪永遠是溫馨的，不會因漏天而看到家變破敗。
實作備註：開門過場＝一次性 CSS/SVG 過場（記 localStorage 或 User flag「已看過居家進場」避免重播）。溫馨度條＝完成數推算的階級（同 journeyWorld 的 tier 概念，純衍生）。「今日提示」＝算「離下一件家具解鎖還差幾次」。皆零額外 schema（除既有 activeWorld）。
