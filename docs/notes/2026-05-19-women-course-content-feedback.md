## 📌 需要釐清的 3 點

### 1. 累加還是替換？(影響架構)

每個 type 從 L1 → L2 → L3 → L4，當使用者升到 L4 時，L1/L2/L3 的 task 是：

- (a) **累加**：L4 使用者同時做 L1+L2+L3+L4 全部 task
- (b) **替換**：L4 使用者只做 L4 的 task，L1-L3 變成「已畢業」不再追蹤

**用雛菊型飲食路徑舉例：**

| Phase | Task |
|---|---|
| L1 | 早上喝兩大口水 (200cc) |
| L2 | 每餐前先喝一杯水 |
| L3 | 下午喝完 500cc |
| L4 | 早上 500cc / 中午 500cc / 晚上 500cc |

- 如果累加 → L4 同時做：兩大口 + 餐前一杯 + 下午 500 + 早中晚各 500 → **總水量會超量、且 task 一天太多**
- 如果替換 → L4 只做「早中晚各 500cc」→ 較合理

從內容直覺看比較像「**替換**」，但需要你明確說明。

### 2. 生理期 task 擴充建議

目前每個 type 只有 1 個生理期 task，建議擴到 2-3 個讓使用者在生理期那 5 天有更多選擇。建議內容方向（你可自行決定）：

**雛菊型**（穩定保養）
- 飯後散步 10 分鐘 ✓ 既有
- 多喝溫水（避免冰飲）
- 早睡 30 分鐘

**玫瑰型**（週期波動）
- 溫和伸展 ✓ 既有
- 避免高鹽食物（水腫管理）
- 不安排高壓會議（情緒緩衝）

**蘭花型**（週期不規律）
- 熱敷小腹 ✓ 既有
- 提早 30 分鐘睡覺
- 喝一杯熱薑茶（暖宮 + 安撫）

**向日葵型**（代謝能量）
- 想吃甜食先吃堅果 ✓ 既有
- 補充 Omega-3 食物（鯖魚、奇亞籽、核桃）
- 避免血糖大起大落（少量多餐）

### 3. 雛菊型水分方法論統一

L2「每餐前一杯水」是**行為觸發型**（cue-based），L4「早中晚各 500cc」是**計量目標型**（quantity-based）。兩種方法論都好，但同一個 type 同一條路徑應該選擇一種：

- **選 (a) cue-based**：L1 早上、L2 餐前、L3 下午、L4 補上晚餐前一杯（每天 5+ 次「觸發補水」）
- **選 (b) quantity-based**：L1 早上 200cc、L2 加午餐 200cc、L3 下午 500cc、L4 早中晚各 500cc 共 1500cc

混用會讓使用者升到 L4 時邏輯不連貫。建議擇一。
