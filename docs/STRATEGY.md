## 1. 目標用戶 & 問題

- **問題**：習慣養成很難——多數習慣 App 靠提醒 + 連續打卡（streak），用恐懼/懲罰驅動，漏一天就崩、留存差。
- **目標用戶**【待定，建議】：想長期改善健康/生活、但「自律靠意志力」一再失敗的人；對「有科學根據、不靠罪惡感」的方法有共鳴者。與 cofit 既有營養/DTx 用戶高度重疊。
- **與 cofit 的關係**【待定】：habitnext1 為 cofit 生態的「習慣養成模組」，可獨立獲客、亦可與 cofit 營養服務交叉銷售。

## 2. 價值主張（為什麼選我們）

- **行為科學為核心**：BJ Fogg 錨點/Focus Map（用 Impact×Ability 挑「黃金行為」）、Atomic Habits 身分認同（「我是個…的人」）、GENESIS+IO 九大領域。
- **零懲罰設計**：珍惜驅動而非恐懼驅動——漏天不崩、不羞辱。
- **科學佐證透明**：每個習慣附證據力指標，建立信任（呼應 cofit 專業形象）。
- **延遲滿足的收藏層**（持續精進中）：把累積渲染成「越蓋越美的城市 / 會長大的公仔」——留存引擎，非核心功能。

## 3. 商業模式 / 變現【待定 · strawman】

建議 **Freemium + 達人計劃飛輪**：

| 層 | 內容 | 變現 |
|---|---|---|
| **免費** | 核心習慣追蹤、嚮往/Focus Map、基本模板、統計 | 獲客、留存 |
| **付費**（訂閱 or 單次解鎖）| **達人計劃**（KOL/專家設計的結構化計劃 + 親簽徽章）、進階模板、進階遊戲化外觀 | 主要營收 |

**成長飛輪（達人計劃）**：105 萬訂閱級 KOL/專家設計計劃 → 用戶完成獲「親簽徽章」→ 截圖炫耀＝免費行銷 → 專家也有動機推廣（品牌曝光）→ 免費/付費分層變現 → 全球專家＝護照式收集。

> 待 PM 拍板：訂閱 vs 單次？價格帶？分潤模式？是否賣外觀（cosmetics）？與 cofit 既有付費的關係？

## 4. 北極星指標（North Star）【待定 · strawman】

建議：**「每週完成 ≥N 個習慣的活躍用戶數」（Weekly Active Habit-Completers, WAHC）**

- 為何選它：直接反映「用戶真的在養成習慣」（產品的真價值），而非下載數/開啟數等虛榮指標；同時與留存、營收正相關。
- `N`【待定】建議先設 **3**（一週至少完成 3 次習慣＝有在用）。

## 5. 關鍵指標框架（AARRR + 護欄）

| 階段 | 指標 | strawman 目標【待定】 |
|---|---|---|
| **獲取 Acquisition** | 註冊數、來源別 CAC | 追蹤來源（KOL / 自然 / cofit 導流）|
| **啟用 Activation** | 註冊後 D1 內「設定第一個習慣/計劃 + 完成第一次」比例 | ≥ 40% |
| **留存 Retention** | D7 / D30 留存、週完成率、平均 streak | D7 ≥ 30%、D30 ≥ 15% |
| **營收 Revenue** | 付費計劃轉換率、ARPU、達人計劃完成率 | 免費→付費 ≥ 3-5% |
| **推薦 Referral** | 徽章/成果分享數、邀請轉換 | 追蹤分享率 |
| **護欄 Guardrail** | 不為「成癮式黏著」而犧牲零懲罰原則；隱私（座標/照片 opt-in）；通知不轟炸 | 投訴率 / 退訂原因監測 |

## 6. v1 上線成功標準【待定 · strawman】

上線後 4–8 週要看到：
1. **啟用率** ≥ 40%（註冊者完成第一個習慣）。
2. **D7 留存** ≥ 30%。
3. **週完成率** 中位數 ≥ 3 次/週（＝北極星有動能）。
4. 無 P0 事故（資料安全/外洩、資料遺失）——對應 `HANDOFF.md` §4 的工程 blocker 必須先清。
5. 質性：使用者訪談能說出「這跟其他習慣 App 不一樣，因為…（科學/不懲罰）」。

> 數字是 strawman，請依你的市場與資源調整。建議上線前在分析工具（GA4 / PostHog / Amplitude 擇一）把 §5 指標埋好。
